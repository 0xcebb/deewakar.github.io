Headlines

Main Headline starts with *, subheadings are created by adding more asterisks.

Show/Hide contents

TAB will cycle local subtree, Shift-TAB cycles through states globally.

Links to other places

Hyperlinks are of the format [[Link] [description]]

e.g. Org mode tutorial

e.g. anther org file

Internal links are written as [[ target ] ] and dedicated targets are marked with angular brackets

e.g. go to the TOP No description for this link

Other external links

Files are linked with `file` e.g. `./sample.txt` Links can be visited with C-c C-o. external links documentation

Ordered and unordered lists

ordered lists have numbers

  1. first
  2. second
  3. third

unordered lists start with '-', '+' or '*'

  • blabla
  • foo
  • bar
  • baz
  • fizzbuzz

Blocks

Blocks are enclosed inside "#+BEGIN" and "#+END" 

(defun org-xor (a b)
    "Exclusive or."
    (if a (not b) b))

The source blocks can be edited in native mode with C-c '.

Tables

field1 filed2 field3
John 32 Teacher
Anna 28 clerk
Benjamin 26 Student

Press C-c C-c to realign the table automatically

Images

toggle inline image view with C-c C-x C-v (org-toggle-inline-images)

Math symbols and equations

subscripts with _ and superscripts with ^

  • The speed of light is 3 x 108 m/s
  • let f0 be the the initial function.

Toggle WYSISYM mode with C-c C-x \

Latex-like symbols

  • The circumference of a circle is given by 2πr
  • As α → 0, γ → ∞

Toggle WYSISYM mode with C-c C-x \

Latex fragments

\begin{align} x&=\sqrt{b} \\ y&=x^2 \\ \end{align}

Inline latex

Latex code can be embedded inline by enclosing inside '\( \)' or '\[ \]' We're given, \(E = mc^2\) and \(T = 2\pi\sqrt{\frac{l}{g}}\)

preview latex fragments on point with C-c C-x C-l Note: make sure you have latex (texlive) and dvipng installed
